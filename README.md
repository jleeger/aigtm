# AIGTM Platform

AI-native, modular, feed-forward Go-to-Market operating system.

## Structure
- `/docs` — architecture documents, module specs, BCO schema spec
- `/schema` — BCO database schema (PostgreSQL SQL files)
- `/orchestrator` — Orchestrator scaffold code
- `/connectors` — Connector Registry and individual connectors
- `/modules` — one folder per module (m00 through m18)
- `/solution-sets` — solution-set configuration files
